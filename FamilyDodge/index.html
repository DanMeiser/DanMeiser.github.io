GAME NOT YET EXPORTED

Export the Godot project as a Web build and the game will appear here automatically.

  1. Install Godot 4.3+ (standard, not Mono/C#)

  2. Open Godot → Import → select FamilyDodge/project.godot

  3. Go to Project → Export — the Web preset is already configured

  4. Click Export Project (not “Export PCK/ZIP”) → the output path is pre-set to game/index.html

  5. Commit the FamilyDodge/game/ folder and push to GitHub Pages

GitHub Pages note: Godot 4 web exports require SharedArrayBuffer, which needs COOP/COEP headers. GitHub Pages does not set these by default — the game will run locally (npx serve .) but may show a black screen on GitHub Pages. To fix this on GitHub Pages, add coi-serviceworker to the exported game/index.html.

FAMILY DODGE

► Dodge the Creeps – Family Edition ◄

Godot 4 Desktop App

HOW TO PLAY

Pick Calvin, Bailey, or Lilly as your character. The other two become homing creeps — and more spawn as time goes on. Survive as long as you can!

W / ↑Move up
S / ↓Move down
A / ←Move left
D / →Move right

SETUP

  1. Install Godot 4.3+ (standard, not Mono/C#)
  2. Open Godot → Import → select FamilyDodge/project.godot
  3. Let the editor finish importing assets, then press F5 to play

Assets are already pre-copied into FamilyDodge/assets/.
